# Python-11-small-Project

A collection of 11 small Python projects demonstrating various programming concepts and applications.

## Projects

### 1. ATM_PIN_SYSTEM
A simple ATM PIN verification system with features:
- PIN verification and change
- Balance checking
- Withdraw and deposit operations
- Maximum attempt security

**Run:** `python ATM_PIN_SYSTEM/atm_pin_system.py`

### 2. Calculator
A calculator with basic and advanced arithmetic operations:
- Addition, subtraction, multiplication, division
- Power and square root calculations
- Modulo operations
- Calculation history

**Run:** `python Calculator/calculator.py`

### 3. EMAIL_Transfer
An email sending simulation system:
- Compose and send emails
- Inbox, sent, and drafts folders
- Email validation
- Read and delete emails

**Run:** `python EMAIL_Transfer/email_transfer.py`

### 4. File_Transfer_App
A file transfer and management application:
- List files and directories
- Copy, move, rename, and delete files
- Create directories
- Transfer history logging

**Run:** `python File_Transfer_App/file_transfer_app.py`

### 5. Flask_API_Project
A RESTful API using Flask framework:
- CRUD operations for tasks
- GET, POST, PUT, DELETE endpoints
- Filter completed/pending tasks
- JSON responses

**Run:** `pip install flask && python Flask_API_Project/flask_api.py`

### 6. Movie_Ticket
A movie ticket booking system:
- View movies and show times
- Visual seat map with seat types
- Book and cancel tickets
- Printable ticket generation

**Run:** `python Movie_Ticket/movie_ticket.py`

### 7. Taju_AI_Chat
A simple AI chatbot simulation:
- Natural language responses
- Jokes and fun facts
- Conversation history
- Time and date queries

**Run:** `python Taju_AI_Chat/taju_ai_chat.py`

### 8. Positive_Negative_Checker
A number classifier application:
- Check if numbers are positive, negative, or zero
- Multiple number checking
- Statistics and history
- Even/odd checking

**Run:** `python Positive_Negative_Checker/positive_negative_checker.py`

### 9. Streamlit_Dashboard
A data visualization dashboard:
- Interactive charts (line, bar, area)
- Key metrics display
- Data filtering controls
- Statistical summaries

**Run:** `pip install streamlit pandas && streamlit run Streamlit_Dashboard/streamlit_dashboard.py`

### 10. Text_To_Speech
A text to speech converter:
- Speak text aloud
- Save audio to file
- Voice selection
- Rate and volume control

**Run:** `pip install pyttsx3 && python Text_To_Speech/text_to_speech.py`

### 11. Tic_Tac_Toe_Game
A classic Tic-Tac-Toe game:
- Two-player mode
- Play against AI
- Score tracking
- Strategic AI opponent

**Run:** `python Tic_Tac_Toe_Game/tic_tac_toe.py`

## Requirements

Basic requirements for all projects:
- Python 3.6+

Additional requirements for specific projects:
- Flask API: `pip install flask`
- Streamlit Dashboard: `pip install streamlit pandas`
- Text to Speech: `pip install pyttsx3`
